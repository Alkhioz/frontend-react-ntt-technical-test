# Prueba tecnica NTT Data

## Ejecutar backend

```
cd /repo-interview-main
npm run start:dev
```

## Ejecutar frontend

```
npm run dev
```

## Ejecutar pruebas unitarias

```
npm run test
```


## Ejecutar pruebas con coverage

```
npm run coverage
```

## Ver los componentes en storybook

```
npm run storybook
```
